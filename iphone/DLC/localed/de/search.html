App guide

1. In Search tap the Scan at the top of the screen (must be within range) to find Sensors.

2. Tap the required Sensor and enter the Passcode, for a new Sensor, enter 00000. If a new Sensor then you are required to change the Passcode (Recommend using a ‘Building’ passcode).

3. In Status tap the Dim Level to ensure you are connected to the correct Sensor

4. In Search, tap Edit Device Name and change the name to match the room/area.

5. Conduct an Occupancy Walk Test in Status Screen the PIR trigger will change. Red for Absence or Green for Presence.

The Sensor is now in operational mode. Advanced set-up an now be completed. For further details refer to instruction sheet.
