Relay Channel 3 Settings Help

You can change the behaviour of the relay from this page.

Features: (Touch for more information)

Manual Control Only

The channel will only respond to manual switch control and will not be controlled by the PIR and Timers.

PIR/Timer/Manual

Responds to presence detection from the PIR, will turn off automatically when no presence is detected and can be manually controlled.

DALI Power Save Mode

Enable this mode when using the relay to control the mains power to DALI luminaires connected to iDim Orbit for additional energy savings. The manual switch input for the Relay is disabled in this mode. The relay turns on whenever either DALI channel is on and turns off 30 seconds after both channels are at 0%.

Time Clock Control

This mode provides automatic time control of the relay. It turns the relay on or off at the start of Profile 1 and Profile 2. Manual control of the relay is still possible in this mode.
