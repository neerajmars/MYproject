Profile Settings Help

Use this page to change the way the iDim Orbit behaves and save settings as Applicaton Files. There are two behaviour Profiles available which can operate during two different periods in a day. To use both Profiles, please enable the Scheduler. If the Scheduler is disabled, only Profile 1 will be used.

Features: (Touch for more information)

Profile 1/Profile 2

Select which Profile to view and edit.

Light vs Time Diagram

For reference only. Shows the relationship of the 2 stage sensor timeout:

  1. On Level with Occupancy Timeout
  2. Power Save Light Level with Transition Timeout

Levels and Timeouts Display

Shows the currently selected levels and timeout periods for the selected Profile. Touch the settings button to edit.

Constant Light Level Adjust

Shows the currently stored constant light target level. Use the + and - buttons to manually adjust the target level. Press Sync to save the new level.

Absence Mode

When this is turned off, the sensor behaves in presence mode. When turned on, it is in absence mode. Presence = auto on by PIR, auto or manual off Absence = manual on by switch input, auto or manual off.

Constant Light

Use this to enable and disable constant light for each Profile. Please refer to the Live Status Help Page for more information.

Bright-Out

Bright-Out stops lights being turned on if ambient light is above the constant light target level. It can only be used in Presence Mode and if constant light is active.

Save File

Save customised Profile settings for back-up and sharing with other users. Insert a Filename (please do not use spaces), Application File Heading and a brief description of the settings. Press save and they will appear in the Custom Application Files section of the Manage Files page.
