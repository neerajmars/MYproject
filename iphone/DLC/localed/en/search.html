Use this page to search for iDim Orbits, connect, disconnect, edit device names and pass codes. Please ensure that your device's Bluetooth is enabled. 

Features: (Touch for more information)

Search

  • Touch the search button to find nearby iDim Orbits. Can also be used to disconnect once a connection has been made. 

Filter by signal strength

  • Use this to control the number of devices displayed in the search results.
  • Move the slider all the way to the right to only display iDim Orbits with the strongest signal strength.
  • Move the slider all the way to the left to show all iDim Orbits with a signal strength strong enough to make a reliable connection.
  • A scan is initiated when the slider is moved to a new position. 

Search results

  • Shows all devices within range (filtered).
  • Touch on any device to make a connection.
  • The default pass code is 00000. When a connection is made, the green LED on the front of iDim Orbit will start flashing.
  • Enter a new pass code when prompted.
  • Alter the pass code and device name by returning to this page when connected.
  • A blue tick appears next to the connected iDim Orbit.
  • Touch the connected iDim Orbit again to disconnect.  
  • Please contact your local Helvar representative if the pass code has been changed and lost.
