The iDim Orbit app allows simple configuration of Helvar's iDim Orbit OB-1101 and OB-1501 products via a Bluetooth® connection. A range of lighting control applications can be easily set up using the intuitive user interface. Choose from a library of standard Application Files for offices and classrooms or create custom Application Files for project specific settings. Compatible with devices with Bluetooth 4.0 or greater:

  • iPhone 4S onward 
  • iPad 3rd generation onward
  • iPad Mini first generation onward 
  • iPad Air first generation onward
  • iPod Touch 5th generation onward
