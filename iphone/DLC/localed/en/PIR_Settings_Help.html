PIR Settings Help

This page enables settings related to the PIR to be modified.

Features: (Touch for more information)

Exit Delay

This enables channels to be manually turned off in Presence Mode. iDim Orbit will not allow channels with PIR/Timer/Manual enabled to be triggered on by the PIR after being manually turned off while movement is still being detected. Only when no movement has been detected for the length of the Exit Delay will lights be triggered on by the PIR.

Enable/Disable Movement Detectors

Use this to modify the detection pattern of five sensor variants by turning off side facing PIRs. Touch the side facing PIRs on the image. Green is enabled, red is disabled.
