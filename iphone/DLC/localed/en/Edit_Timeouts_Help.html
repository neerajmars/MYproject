Edit Timeouts Help

Use this page to adjust the levels and timeout periods.

Features: (Touch for more information)

On Level

This enables channels to be manually turned off in Presence Mode. Sets the level at which the DALI outputs will come on to when activated by the PIR or manual switch. Use the slider to change the level or touch on the value to type in a particular level.

Occupancy Timeout

Determines how long the DALI channels linked to the PIR/Timer will stay at the On Level when no movement is detected. Use the slider to change the time period or touch on the value to type in a particular time.

Power Save Light Level

This is the level that the DALI outputs linked to the PIR/Timer will go to at the end of the Occupancy Timeout. Use the slider to change the level or touch on the value to type in a particular level.

Transition Timeout

Determines how long the channels linked to the PIR/Timer will stay at the Power Save Light Level for during the second timeout stage. Use the slider to change the time period, or touch on the value to type in a particular time. Select 0 mins to disable the second timeout stage or Infinity to stop the outputs from ever going off automatically.
